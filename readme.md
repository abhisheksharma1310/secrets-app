## Web App Installation Instructions

- Clone secrets project
- Go to the project directory

```
git clone https://github.com/abhisheksharma1310/secrets-app
cd secrets/
```
- Install all packages 

- Run web app

```
npm install
npm run start
```

- For build package

```
npm run build
```
